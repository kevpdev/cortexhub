Wrapper for `~/.ai-core/scripts/capture.sh`.

## Steps

1. Take the note passed with the command.
   - If no note provided: ask "What do you want to capture?"

2. Run in terminal:

```bash
~/.ai-core/scripts/capture.sh "note text"
```

3. Display the script output (one line confirmation).
